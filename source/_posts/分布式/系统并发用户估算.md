`泊松分布`并发用户与最大并发用户计算：

##### 1.平均并发用户数
C = n*L/T  
-  n 为login session数  
-  L 为login session平均时长  
-  T 为考察时长

##### 2.并发峰值用户数
C'≈ C + 3*√200

##### 3.吞吐量
F=VU * R / T  
- F为吞吐量
- VU表示虚拟用户个数
- R表示每个虚拟用户发出的请求数
- T表示性能测试所用的时间
- R = T / TS

例：假设有一个OA系统，该系统有3000个用户，平均每天大约有400个用户要访问该系统，对一个典型用户来说，一天之内用户从登录到退出该系统的平均时间为4小时，在一天的时间内，用户只在8小时内使用该系统。  

根据公式计算得：  
C = 400 * 4/8 = 200  
C'≈200+3 * √200 = 242
